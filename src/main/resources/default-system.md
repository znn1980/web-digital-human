# 酒店管理AI提示词（全场景适配）
说明：以下提示词分4大类，适配AI生成方案、话术、流程、方案优化等需求，可直接复制使用，根据酒店规模（高端/中端/快捷）、核心痛点调整细节即可，无需额外修改格式。
## 一、通用基础类（适配所有酒店管理场景，快速生成基础内容）
1.  作为资深酒店运营顾问，结合酒店行业规范和客户体验需求，帮我梳理一份通用酒店管理全流程（涵盖前厅接待、客房服务、餐饮管理、安全保障、客户反馈处理），要求流程清晰、可落地，标注每个环节的核心重点和易错点，语言简洁专业，适合酒店新员工培训使用。
2.  以酒店管理者视角，生成一份简洁实用的酒店管理日常工作清单，包含每日、每周、每月核心工作内容，明确工作优先级和执行标准，适配中端商务酒店，助力高效统筹酒店各项运营事务，减少管理漏洞。
3.  结合当前酒店行业趋势（智能化、个性化服务），帮我撰写一段酒店管理核心理念文案，突出“客户至上、高效运营、细节为王”的核心，用于酒店内部培训、员工动员或对外宣传，语言有感染力，贴合酒店服务调性。
## 二、运营管理类（核心痛点解决，适配日常运营优化）
1.  作为酒店运营专家，针对酒店客房空置率高、回头客占比低的问题，结合中端酒店定位，制定一套可落地的运营优化方案，包含客房定价策略、会员体系搭建、老客户召回活动，明确每个方案的执行步骤、预期效果和成本控制，避免空泛建议。
2.  帮我设计一套酒店客房服务标准化话术和流程，涵盖客房清洁、客户需求响应、客房投诉处理、个性化服务（如特殊客群照料），要求话术礼貌得体、流程规范，兼顾效率和客户体验，可直接用于客房员工培训，减少服务失误。
3.  结合酒店智能化趋势，帮我梳理一份酒店智能化管理落地建议，涵盖前厅自助入住机、客房智能控制系统、客户服务机器人的选型、应用场景和运营维护要点，适配高端度假酒店，突出科技感和便捷性，同时控制投入成本。
## 三、服务提升类（聚焦客户体验，优化服务细节）
1.  作为酒店服务顾问，针对酒店前厅接待效率低、客户等待时间长的问题，设计一套前厅服务优化方案，包含接待流程简化、员工分工、应急处理（如高峰时段、客户投诉），要求方案具体可操作，能有效缩短客户等待时间，提升前厅服务满意度。
2.  帮我生成多组酒店客户沟通话术，涵盖客户预订确认、入住引导、需求咨询、投诉处理、离店送别，适配不同客群（商务客、家庭客、度假客），话术亲切自然、专业得体，既体现酒店服务温度，又能高效解决客户需求。
3.  结合酒店餐饮服务，设计一套餐饮管理优化方案，涵盖菜品迭代、食材采购、服务流程、卫生管控，突出“新鲜、美味、便捷”，适配酒店自助餐厅/特色餐厅，兼顾客户口味和成本控制，同时提升餐饮服务口碑。
## 四、营销推广与应急处理类（适配获客和风险管控）
1.  作为酒店营销专家，结合节假日（如国庆、春节、周末），为中端商务酒店制定一套短期营销推广方案，包含线上渠道（OTA平台、微信公众号、短视频）、线下活动（到店礼、套餐优惠），明确推广节奏、宣传文案和转化目标，助力提升客流量和营收。
2.  帮我梳理酒店常见应急场景（如客户突发疾病、设备故障、恶劣天气、安全事故）的处理流程和话术，要求流程规范、话术严谨，明确各岗位职责，能快速响应、妥善处理，最大限度降低客户不满和酒店损失，适合酒店全员培训使用。
3.  针对酒店OTA平台评分低、负面评价多的问题，帮我制定一套评价优化方案，包含负面评价回复话术、客户满意度提升措施、OTA平台运营技巧，要求话术真诚得体，措施可落地，能有效提升平台评分和客户信任度。
